# CivicQuest

## What it is
AI-powered community engagement platform to discover local volunteering events, register seamlessly, learn sustainability through a gamified experience, and chat with an Azure-based bot.

## Key features
- Event discovery + registration
- Admin portal (create/edit/delete events)
- Chat experience for guidance and Q&A

## Tech
Node.js + EJS; Azure bot integration.

## Next improvements
- Location-based recommendations
- Organization verification + profiles
